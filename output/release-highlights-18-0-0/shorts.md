# Shorts/Reels — release-highlights-18-0-0

## HOOK
We deleted twelve thousand lines of code from our admin — and it got better.

## BEAT 1
Shopsys 18.0 ripped out the legacy custom admin theme. Years of bespoke CSS, one-off modals, utility JS nobody wanted to touch. Visual: split screen — old admin on the left, new Tabler UI admin on the right.

## BEAT 2
We replaced it with Tabler UI. Responsive on mobile, standardized form templates, reworked modals. Net result: roughly twelve thousand fewer lines to maintain. Visual: screen recording scrolling through the new admin on a phone, then desktop.

## BEAT 3
Same release also cut GraphQL product detail payloads by seventy to eighty percent, moved order email prep out of checkout, and shipped EU cooling-off withdrawal out of the box. Visual: quick cuts — payload size graph, checkout flow, withdrawal form.

## CTA
Full 18.0 release notes on github.com/shopsys/shopsys — link in bio.
