# Twitter/X — release-highlights-18-0-0

## THREAD

1/ Shopsys 18.0.0 shipped.

2/ Headline nobody will lead with: we deleted ~12,000 lines of legacy admin code.

3/ The old admin was years of bespoke custom theming. Expensive to maintain. Dated on mobile.

4/ Replaced with Tabler UI. Responsive. Standardized form templates. Modals that behave.

5/ A single GraphQL field change cut product detail payloads by 70-80% for store availability data. Mobile feels it.

6/ Order email preparation moved out of the checkout critical path. Faster order creation, fewer checkout failures.

7/ EU cooling-off (14-day withdrawal) is now a first-class feature. Configurable deadline, per-domain instructions, dedicated "Withdrawn" order status.

8/ The best releases look like this. Less code. Lighter payloads. Compliance built in, not bolted on.
